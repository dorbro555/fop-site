## What is Fop?

<CardGrid stagger>
	<Card title="Unified Package Management" icon="pencil">
		Provides a unified interface across Linux distributions.
	</Card>
	<Card title="Attach notes" icon="add-document">
		Zypper doesn't have a friendly UX for attaching memos to package operations. This is where 'fop' comes in, eg. `fop install openssl --notes "for managing certs"`
	</Card>
	<Card title="Zypper under the hood" icon="setting">
		Fop is just a wrapper around zypper, providing the same functionality with added features.
	</Card>
	<Card title="Read the docs" icon="open-book">
		Learn more in [the Fop Docs](/guides/example).
	</Card>
</CardGrid>
