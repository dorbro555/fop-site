## What is Fop?

<CardGrid stagger>
	<Card title="Simplified Package Management" icon="pencil">
		Zypper is a package manager, while fop is a tool that helps manage packages.
	</Card>
	<Card title="Reproducible Installations" icon="setting">
		Declarative package configurations, allowing system upgrades which can be redeployed and rolled back.
	</Card>
	<Card title="Zypper under the hood" icon="setting">
		Fop is just a wrapper around zypper, with added features.
	</Card>
	<Card title="Attach notes" icon="add-document">
		A friendly UX for attaching memos to package operations. eg. `fop install openssl --notes "for managing certs"`
	</Card>
	<Card title="Read the docs" icon="open-book">
		Learn more in [the Fop Docs](/guides/example).
	</Card>
</CardGrid>
